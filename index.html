The self-hosted web IRC client

Always connected.

Install The Lounge from npm:

npm -g install thelounge  
npm -g install thelounge@next

Or try the demo.

Why you should use it

Always Connected

The Lounge never sleeps. Just sign in and pick up where you left off.

Responsive Interface

The client works smoothly on every desktop, smartphone and tablet.

Cross Platform

It doesn't matter what OS you use. The Lounge just works wherever Node.js runs.

Multi-user Support

Supports multiple user accounts. Share your server with your friends!

Open Source

The Lounge is open source and licensed under the MIT license.

Community Managed

It is run by a dedicated community as a fork of Shout. Find us in #thelounge on Freenode.
